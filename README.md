# MagicSea Online Server Status Tracker

**Tool for monitoring online player count in MagicSea Online — the mobile version of the legendary Piratia Online (Tales of Pirates)**

> Download and play: [piratia.online](https://piratia.online)

---

## What is this?

This is a lightweight Python script for tracking the live server status of **MagicSea Online** — a free mobile MMORPG known to many players as **Piratia Online** or **Tales of Pirates**.

The script fetches the current player count directly from the official server, so you can check activity before logging in or track population trends over time.

Useful for:
- Players wondering if the server is alive before downloading
- Community members tracking MagicSea Online player count over time
- Fans of the original Tales of Pirates curious about the mobile revival

---

## About MagicSea Online (Piratia Online)

**MagicSea Online** is the official mobile continuation of the classic **Piratia Online** MMORPG, originally known in Russia as Tales of Pirates. Available free on Android and iOS.

The game preserves the original world of **Ascaron** with cities Argent, Shaitan and Icicle, all original races and classes — now fully playable on mobile.

**Classes:**
| Class | Role | Best for |
|-------|------|----------|
| Voyager | AoE DPS | Dungeons |
| Sharpshooter | Ranged DPS | PvP |
| Seal Master | Control | Debuffs |
| Cleric | Support | Raids |
| Crusader | Melee DPS | Solo |
| Champion | Tank | Any |

**Races:** Lance, Ami, Phyllis, Carsise

**Key features:**
- Free to play, no auto-combat
- PvE dungeons and raids
- Guild wars and PvP
- Russian and English localization
- Stable server 24/7, low ping for Russia and CIS
